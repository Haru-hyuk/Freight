# 🚚 Freight
관리자 웹 + 화주/기사 모바일 앱 구조의 화물운송 플랫폼

---

## 📌 프로젝트 개요
본 프로젝트는 화물 접수부터 배차, 운송 완료까지의 전 과정을 관리하는  
화물운송 플랫폼입니다.

관리자 전용 웹과 화주·기사용 모바일 앱을 분리하여  
운영 효율성과 현장 사용성을 각각 최적화하는 것을 목표로 합니다.

---

## 👥 사용자 구성
- **관리자 (Admin)**
  - 화물 및 기사 관리
  - 배차 및 운송 상태 관리

- **화주 (Shipper)**
  - 화물 접수
  - 배차 / 운송 상태 확인

- **기사 (Driver)**
  - 차량 등록
  - 배차 수락 및 운송 진행

---
## 🧱 프로젝트 구조
-
├─ backend/ # 공통 API 서버 (계층형 구조)
├─ frontend_admin/ # 관리자 전용 웹
├─ mobile_app/ # 화주·기사 통합 모바일 앱
└─ docs/ # 설계 문서 및 자료
---

## 🔄 기본 흐름
1. 화주 화물 접수
2. 관리자 배차 진행
